토큰은 어디에 저장되나요?
암호화된 형태로 Garim 서버에 저장되며, 본인 인스타 점검 외 용도로 사용되지 않습니다. 마이페이지에서 언제든 연결을 해제하면 토큰이 즉시 폐기됩니다.
점검 결과 영상은 자동으로 다시 올라가나요?
아니요. Garim은 인스타에 게시·삭제·수정을 하지 않습니다 (B-2 권한 정책). 결과 영상을 다운로드 받으신 후, 직접 기존 게시물을 삭제하고 새 버전을 업로드하시면 됩니다.
v2 "정기 자동 스캔" 기능은 무엇인가요?
v2에서는 일정 주기로 새 게시물을 자동 스캔해 위험을 알려드립니다. MVP1·v1 정식에서는 수동 점검만 제공됩니다.
