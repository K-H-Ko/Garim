# 🛡️ 프로젝트: Garim (가림)

## **AI 기반 멀티모달 개인정보 도싱(Doxxing) 방지 및 데이터 치환 엔진**

## 1. 프로젝트 배경 및 기획 의도

현대 디지털 사회에서 개인정보 유출은 더 이상 해킹에 의해서만 일어나지 않습니다.<br/>
무심코 올린 브이로그 영상 창문에 비친 간판, 라이브 방송 중 0.1초 노출된 택배 송장,<br/>
야외 방송 중 지인과 통화하며 실수로 말한 동/호수가 결합되어 한 사람의 주거지와 생활 패턴이 특정되는 **퍼즐 도싱**의 시대입니다.

- **도싱(doxxing)** 이란?
  특정인의 신상 정보를 인터넷을 통해 알아낸 뒤 무차별적으로 인터넷에 공개해 버리는 사이버 테러의 일종(신상 털기)

기존의 모자이크나 검은색 박스 처리는 콘텐츠의 몰입도를 심각하게 훼손하여 사용자들이 보안 처리를 기피하게 만듭니다.<br/>
이에 우리는 콘텐츠의 시각적 가치는 100% 보존하면서, 치명적인 개인정보만 감쪽같이 가짜 데이터로 치환해 주는 지능형 프라이버시 엔진을 개발하여,<br/>
누구나 안전하게 일상을 공유할 수 있는 환경을 만들고자 합니다.

## 2. 타겟 고객 및 선정 배경

### 🎯 메인 타겟: 인플루언서, 유튜버, 연예인 등 콘텐츠 크리에이터

- **선정 배경:**<br/>
  이들은 일상을 공유하여 수익을 창출하지만, 역설적으로 스토킹 및 사생활 침해 위험에 가장 크게 노출된 집단입니다.<br/>
  단 한 번의 노출 실수로 거주지를 옮기거나 방송을 중단해야 하는 치명적인 피해를 입습니다.<br/>

- **필요성:**
  이들은 영상의 '미적 퀄리티(영상미)'를 극도로 중요하게 생각하므로, 투박한 모자이크를 꺼립니다.<br/>
  원본과 구별할 수 없는 **'자연스러운 치환'** 기술에 대해 기꺼이 비용(구독료)을 지불할 의사가 있는 고가치 고객층입니다.<br/>

### 👥 서브 타겟: 개인정보 보호가 필요한 일반 대중

- **선정 배경:**
  당근마켓에 영수증을 올리거나 SNS에 브이로그를 올리는 일반인들 역시 잠재적 범죄의 타겟이 됩니다.<br/>
  무료 분석 기능과 Basic 플랜을 제공하여 서비스의 대중적 확산과 트래픽을 확보하는 기반이 됩니다.<br/>

## 3. 서비스 목적 및 차별성

### 🚨 실제 피해 사례 (해결하고자 하는 문제)

1. **영상/이미지(2025.10 발생):**<br/>
   구독자 80만 명의 유명 브이로그 유튜버 A씨. 언박싱 라이브 중 뒤쪽 모니터 유리에 0.1초간 반사된 택배 송장(이름, 집주소)이 유포되어 심각한 스토킹 피해 및 방송 중단 발생.<br/>

2. **오디오/음성(2025.05 발생):**<br/>
   대형 게임 스트리머 B씨 야외 방송 중, 지인과 통화하는 소리(STT)가 마이크에 수음. 특정 아파트 동/호수가 그대로 송출되어 주거지 노출 소동 발생.<br/>

3. **문서(2025.11 발생):**<br/>
   프리랜서 디자이너 C씨의 포트폴리오 PDF 업로드. 화면엔 없었으나 파일 속성(메타데이터)에 이전 고객 실명/계약 금액이 남아 계약 파기 및 소송 발생.<br/>

### 💡 현존 서비스와의 차별성

- **메타데이터 툴(ExifTool):**<br/>
  파일 속성(GPS 등)만 지울 뿐, 영상 화면 속 글자는 지우지 못함.<br/>
  ➡️ **Garim은 화면 속 픽셀과 오디오를 직접 분석함.**

- **기업용 DLP(MS Purview):**<br/>
  사내망 기밀문서 전용, 개인 영상/오디오 처리 불가. 문서(PDF, DOCX)만 가능<br/>
  ➡️ **Garim은 개인의 영상, 사진, 문서, 오디오 전 영역 가능.**

- **일반 영상 편집기(모자이크):**<br/>
  수동으로 영역을 지정해야 하고 결과물이 투박함.<br/>
  ➡️ **Garim은 AI가 자동 탐지하고, 원래 폰트와 조명을 계산해 '가짜 데이터'로 자연스럽게 합성함.**

- **서버 유출 방지:**<br/>
  분석을 위해 원본을 LLM 서버로 보내는 타 서비스와 달리, "주소 형식만 줘"라고 익명 요청하여 서버 유출 원천 차단.

## 4. 핵심 서비스 기능 및 구현 방안

우리 서비스는 크게 **[파일 치환 파이프라인]**과 **[SNS 계정 검증 파이프라인]** 두 가지 독립된 서비스 페이지로 운영됩니다.

### 🟢 [기능 1] 지능형 리스크 진단 리포트 (분석 페이지)

사용자가 문서/이미지/영상을 업로드하면, 메타데이터 치환을 즉시 수행하지 않고 **'분석 리포트'**를 먼저 제공합니다.

- **구현 방안:**
  - **YOLOv11 & SAM 2:** 시각적 객체(송장, 번호판 등) 탐지.<br/>
  - **Whisper & KoELECTRA (딥러닝모델):**<br/>
    영상 내 오디오를 STT로 변환하여 개인정보 문맥 식별.<br/>
    (딥러닝은 개인정보 식별데이터 학습예정, 가벼운 small 모델로 테스트해보고 학습이 잘 안되면 base모델 사용예정)
- **사용자 UX:**<br/>
  해당부분 프레임 이미지와 **"00분 15초: 택배 송장(주소 노출 위험)"** 형태로 시각적 가이드를 제공하여 사용자가 위험을 직관적으로 인지하게 함.

### 🟢 [기능 2] 상호작용형 보안 처리 요청 및 치환 옵션 (편집 제어)

리포트 하단에 실제 적용될 보안 처리 예시(Sample 가이드)를 보여주고, 사용자가 직접 [보안 처리 요청] 버튼을 눌러 제어권을 행사합니다.

- **선택형 메타데이터 치환 (사용자선택 or Random):**
  - **랜덤 진행:**<br/>
    LLM 을 통해 원본과 동일한 형식/길이의 가짜 데이터 자동 생성.<br/>
    (LLM 에게는 사용자 개인정보가 직접 전달되지 않으며, 어떠한 형식으로만 만들어 달라고 요청됨.)
  - **사용자 지정 진행:** 사용자가 직접 변경할 글자를 입력.<br/>
    (단, 합성의 자연스러움을 위해 **원본 글자 수와 동일하게 입력하도록 UI/UX 단에서 유효성 검사 강제**).
- **시각적 합성(STE):** Scene Text Editing 모델을 사용하여, 선택된 가짜 데이터를 원본 영상의 폰트, 굵기, 조명, 기울기에 맞춰 렌더링.
- **오디오 마스킹:** Forced Alignment로 찾아낸 정확한 ms 단위 좌표에 1000Hz 삐- 소리 또는 묵음 합성.

### 🟢 [기능 3] 구간 다운로드 및 재인코딩 (출력 단계)

보안 처리가 완료된 영상을 사용자가 다운로드할 때 효율성을 극대화합니다.

- **전체 다운로드:** 치환이 적용된 전체 영상을 재인코딩하여 제공.
- **구간 다운로드:** 사용자가 숏폼이나 특정 구간만 필요로 할 경우, 텍스트 인풋으로 `시작 시간(예: 01:10) ~ 끝 시간(예: 02:00)`을 입력받음.
- **구현 방안:** 백엔드에서 **FFmpeg**를 활용하여 지정된 구간만 잘라내어(Trim) 재인코딩 수행. 서버 부하를 줄이고 사용자 대기 시간을 단축.

### 🟢 [기능 4] SNS 계정 개인정보 검증 서비스 (독립 페이지로 가능시 구현 / 가능한지 확인 예정)

기존에 업로드된 파일의 위험성을 사후 검증하는 플랫폼 연동 기능입니다.

- **기능 개요:** 별도의 메뉴에서 사용자가 자신의 **인스타그램(또는 특정 SNS) 홈 URL**을 입력.
- **작동 프로세스:**
  1. API 또는 크롤러를 통해 해당 계정의 최근 이미지/영상 게시물을 순차적으로 수집.
  2. Garim의 Vision/Audio 엔진을 백그라운드에서 동일하게 가동.
  3. **결과 리포팅:** <br/>
     "게시물 날짜 25.10.12 - 이미지 내 자동차 번호판 노출", <br/>
     "게시물 날짜 25.11.01 - 캡션 및 영상 내 카드번호 노출" 등 계정 전체의 위험도를 스캐닝하여 리스트업.<br/>
- **목적:** 사용자가 과거에 무심코 올린 데이터의 위험성을 깨닫게 하여, 향후 파일 업로드 전 반드시 우리 서비스를 거치도록 유도하는 역할.

## 5. 최종 결과물 및 기대 효과

1. **완벽하게 세정된 미디어 에셋:** <br/>
   시청자는 마스킹된 가짜(메타데이터) 정보로 치환된, 안전한 파일을 얻게 됩니다.
2. **크리에이터의 편집 워크플로우 혁신:** <br/>
   일일이 프레임을 단위로 모자이크를 추적(Tracking)하고 오디오를 자르던 수십 시간의 편집 노동이 단 3~5분 내외의 원클릭으로 해결됩니다.
3. **도싱(Doxxing)의 원천 차단:** <br/>
   단일 파일뿐만 아니라 SNS 계정 전체의 리스크를 스캐닝함으로써, 파편화된 정보가 모여 위치가 특정되는 퍼즐 매칭 범죄를 사전에 방지합니다.

## 6. 서비스 확장성

1. **B2B(기업) 확장성:**<br/>
   일반인의 주소뿐만 아니라, 특정 기업의 기밀문서 포맷, 사원증 패턴, 소프트웨어/게임의 내부 시리얼 넘버를 집중 학습하여 사내망 정보 유출 방지 솔루션으로 납품.
2. **모바일 앱(App):**<br/>
   향후 App으로 출시하여, iOS/Android의 모바일 앱에서 '공유하기'로 카톡, 메세지, 메일에 사진 공유시 개인정보 탐지 및 보안 마스킹 제공.
3. **통합 디지털 지문 대시보드:**<br/>
   SNS 스캐닝 기능을 고도화하여 유튜브, X 등 여러 플랫폼에 흩어진 사용자의 '디지털 풋프린트' 통합 위험도를 점수화(Scoring)하여 보여주는 구독형 관리 서비스로 진화.

## 7. 결론: 왜 Garim인가?

콘텐츠의 질이 곧 수익인 시대입니다. <br/>
기존의 보안 툴들은 정보 보호를 명목으로 콘텐츠의 미적 가치를 훼손하고 시청자의 몰입을 방해하는 방향 이었다면,

**Garim은 보안과 퀄리티를 기술로 해결 합니다.** <br/>
사용자는 본인이 직접 리스크를 눈으로 확인하고(진단 리포트), 원하는 형태로 정보를 바꾸며(치환 옵션), <br/>
원하는 구간만 빠르게 추출(구간 인코딩)할 수 있습니다. 더 나아가 자신의 SNS 전체를 진단받을 수도 있습니다.

우리는 단순한 모자이크 툴을 만드는 것이 아닙니다. <br/>
창작자들이 보이지 않는 위협으로부터 해방되어 마음껏 일상을 기록하고 공유할 수 있도록 돕는 **디지털 방패'** 입니다.
