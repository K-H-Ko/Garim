솔루션: 지능형 탐지와 감쪽같은 보호

"지우는 것이 아니라, 존재한 적 없던 것처럼 재구성합니다."

1. AI 정밀 분석 (오디오 포함)

영상 속 텍스트는 물론, 오디오를 텍스트로 변환(STT)하여 실수로 노출된 전화번호나 개인정보까지 완벽히 잡아냅니다.

2. 가짜 데이터 치환

원본의 폰트와 조명을 유지한 채 가짜 정보로 교체합니다.
예: 010-1234-5678 ➡️ 010-9876-5432

3. 블러 예외 처리

영상 텍스트가 심하게 흔들려 가짜 데이터 합성이 어색할 경우, 가장 잘 어울리는 자연스러운 블러로 자동 전환합니다.

직관적인 UI 워크플로우

  • Step 1: 분석 페이지에서 검출된 영역 박스 직접 확인
  • Step 2: 보안 처리 샘플 및 가이드 확인
  • Step 3: 하단 보안 처리 요청 버튼 클릭 한 번으로 즉시 해결!

핵심 기술 명세: AI가 파일을 지키는 방법

보호 범위: 풍경/건물을 통째로 가리는 비현실적 기술이 아닙니다. 도용되어 피해를 주는 정확한 개인정보(텍스트/음성)만을 핀포인트로 탐지하여 방어합니다.

👁️ 시각 엔진 (Vision & STE)

사진/영상 속 글자를 지우고 원래 폰트로 가짜 정보를 합성합니다.

  • 탐지 : YOLOv11 + SAM 2로 영상 내 객체와 송장 등의 픽셀 영역을 분리
  • 인식 (OCR): EasyOCR을 통해 텍스트 값 추출
  • 치환 : STE(Scene Text Editing) 및 Inpainting 모델로 원본의 폰트, 조명, 기울기를 계산하여 감쪽같이 가짜 텍스트 합성

🎧 음성 엔진 (Audio Redaction)

실수로 말한 개인정보를 묵음이나 삐- 소리로 1ms 단위로 덮습니다.

  • 듣기 (STT): OpenAI Whisper를 통해 음성을 텍스트로 변환
  • 분석 (NER): KoELECTRA (파인튜닝) 모델이 문맥을 파악해 전화번호, 주소 등 개체명 식별
  • 추적/합성: Forced Alignment 기술로 파형과 텍스트의 좌표를 동기화하여 특정 구간만 정밀 필터링

🔒 웹 서비스 철통 보안 (Zero-Data Prompting)

진짜 정보를 LLM 서버로 절대 전송하지 않습니다. 로컬 엔진에서 "이런 형식의 가짜 번호 하나 만들어줘"라는 익명 요청(Faker 라이브러리 연동)만 주고받아, 처리 중 정보 유출 확률을 원천 차단합니다.
