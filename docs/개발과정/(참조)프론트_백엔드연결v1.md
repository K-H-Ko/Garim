## ★ Phase 2 설계 — 자동 파이프라인 + 페이지 재설계

> 목표: 사용자가 파일 업로드 → 시스템이 자동으로 분석 → 결과 제공. 수동 개입 없음.

### 전체 자동화 흐름

```
[브라우저] 파일 업로드
     │
     ▼
[백엔드] analysis Job 큐 등록 (status: queued)
     │
     ├──────────────────────────────────┐
     ▼                                  ▼
[local_worker.py]              [colab STT 워커] (선택)
OCR 분석 (로컬 CPU)            음성 STT 분석 (Colab)
     │                                  │
     └──────────┬───────────────────────┘
                ▼
         backend_json_merger
         result.json + 상세보기 생성
         /worker/jobs/{id}/results/* 등록
         /worker/jobs/{id}/complete
                │
                ▼
[백엔드] status: completed
                │
                ▼
[브라우저] ③ AnalysisReport (건수+위험도만 표시, 크레딧 차감 없음)
     │
     ▼ "상세보기" 클릭 → POST /detail-access (이미지 2크레딧 / 영상 3크레딧 차감)
     │
     ▼
[브라우저] ④ 상세 페이지 (이미지/영상 분기, bbox 오버레이, 선택)
     │
     ▼
[브라우저] ⑤ 미리보기 (가림바 비교 슬라이더)
     │
     ▼
[마스킹 워커 (Colab or 로컬 더미)]
colab_pipeline_mask.py 자동 polling
     │
     ▼
[백엔드] processed_files 등록 + complete
     │
     ▼
[브라우저] ⑥ 결과 페이지 (미리보기 + 다운로드 + 만료일)
```

## 주의사항 / 알려진 이슈

### 1. timeline_markers id 불일치 가능성
- `timeline_markers[].id` = result.json의 `pii_id` (파이프라인 내부 ID)
- `detections[].detection_id` = DB INSERT 시 새로 생성된 UUID
- 두 값이 달라 마커 클릭 → 카드 스크롤이 미동작할 수 있음
- 해결: `get_job_detections()` 응답에 `pii_id` 필드 포함 또는 프론트에서 매핑

### 2. Download.jsx → ResultPage.jsx 전환 시 App.jsx 라우트 수정 필요
- 기존: `/processing`, `/download` 경로 → 신규: `/result` 경로 추가
- `App.jsx`에서 ResultPage.jsx import 및 라우트 등록 필요

### 3. ~~createAnalysisJob 크레딧 차감 제거 필요~~ ✅ 260615 완료
- `backend/services/analysis.py` 내 `create_analysis_job()` 크레딧 차감 로직 제거 완료
- 차감 시점: 분석 시작 → **상세보기 진입** 시로 이동

### 4. ~~local_worker.py 상시 실행 필요~~ ✅ 260615 자동화 완료
- ~~백엔드 실행 후 별도 터미널에서 `python ocr_mask/local_worker.py` 실행 필수~~
- **`backend/main.py` lifespan 이벤트로 서버 시작 시 자동 백그라운드 스레드 실행**
- `uvicorn main:app` 하나만 실행하면 워커 자동 동작

### 5. 이미지 파일 처리 시 content_type 주의
- processed-file 등록 시 이미지: `"image/jpeg"`, 영상: `"video/mp4"` 정확히 지정 필요

```js
video.addEventListener("timeupdate", () => {
  const frameNo = Math.floor(video.currentTime * 30); // FPS=30
  const boxes = tracks[`frame_${frameNo}`] || [];
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  boxes.forEach(box => {
    ctx.strokeStyle = selected[box.pii_id] ? "red" : "orange";
    ctx.lineWidth = 2;
    const [x1, y1, x2, y2] = box.bbox;
    ctx.strokeRect(x1, y1, x2 - x1, y2 - y1);
    ctx.fillStyle = "red";
    ctx.font = "12px sans-serif";
    ctx.fillText(box.label, x1, y1 - 5);
  });
});
```
