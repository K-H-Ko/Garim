## 1. 프로젝트 전체 흐름 및 현황 요약
- **목표 (무엇을 구현하려 했는가)**: 이미지 내 개인정보(PII)를 주변 배경 훼손 없이 감쪽같이 지우고 가짜 데이터로 치환(마스킹)하는 시스템 구축.
- **초기 시도 및 방향 전환 (무엇을 하려다 실패했는가)**: 
  1. 초기에 AnyText 단일 모델로 텍스트 탐지/합성을 시도했으나 정밀도 및 한국어 지원 한계로 전면 실패.
  2. **(핵심 변경)** 이후 폰트를 찾아 가짜 텍스트를 '합성'하는 3D Warp 방식을 시도했으나, 폰트 매칭 한계로 너무 인위적인 티가 났습니다. 이에 **오늘(6/1)부로 가짜 텍스트 '합성' 기능 코드는 전면 폐기/제거하고, 오직 "텍스트 완벽 제거 및 감쪽같은 원본 배경 복원"에만 100% 집중**하기로 방향을 크게 전환했습니다.
- **현재 구현 방식**: OCR 탐지 ➔ 3대 핵심 측정법(엣지, 분산, 구조선) 기반 배경 복잡도 판별 ➔ 하이브리드 배경 복원(SD Inpainting, LaMa, Seamless) ➔ 2중 보호막 타겟팅 지우기.

## 2. 금일(6/1) 세부 작업 내용 (시간 흐름순 정리)
오늘 진행된 작업은 '텍스트 합성 기능 폐기'라는 큰 결정부터 시작하여, 새로운 배경 복원 모델 테스트와 정밀 마스킹 튜닝으로 이어지는 과정이었습니다.

1. **폰트 합성 기능 전면 제거 및 목표 재설정**
   - 폰트 매칭의 한계와 합성의 이질감 문제로 관련 코드를 모두 제거하고, "지정된 범위에서 텍스트만 완벽히 지우고 배경만 원본처럼 복원한다"는 명확한 목표로 노선을 변경했습니다.
2. **3대 핵심 측정법 및 하이브리드 분기 도입**
   - 배경을 '단순/복잡'으로 구분하기 위해 엣지 밀도, 픽셀 대비 및 분산, 형태학적 구조선 감지라는 3가지 강력한 수치 측정법을 도입하여 하이브리드 자동 분기 시스템을 설계했습니다.
3. **복원 모델(SD, LaMa, Seamless) 결합 테스트 및 모델 최종선정**
   - Stable Diffusion 인페인팅 로컬 다운로드 코드를 작성해 로딩 속도를 개선.
   - SD 모델 단독 사용 시 전체 이미지를 망가뜨리는 문제를 확인 후, 디퓨저/LaMa/Seamless Clone 각각이 가장 잘하는 것을 섞어 쓰는 조합법을 시도하며 발생하는 셰이프 불일치 및 NoneType 에러 해결.
   - Stable Diffusion 모델의 배경 생성시 텍스트를 임의로 생성하는 단점이 발생하여 **LaMa + Seamless** 만 사용하는 것으로 결정.
4. **주변 글자 훼손 방지를 위한 정밀 타겟팅 튜닝**
   - 잔상 없이 지우려다 주변 글자까지 훼손되는 것을 막기 위해, 바운딩 박스 안에서만 정교하게 지우도록 마스킹 범위를 좁히는 디테일 튜닝을 진행.

## 3. 한계점 및 차후 계획
- **현재 고민 중인 문제점 (잔상 이슈)**: 모델 결합과 타겟팅 튜닝 노력에도 불구하고, **일부 텍스트는 아직도 깨끗하게 지워지지 않고 미세한 잔상이나 잉크 찌꺼기가 남는 현상**이 있어, 1픽셀의 오차도 없이 완벽히 삭제 할 방법을 추가로 연구해야 함.
- **차후 계획 (영상 단위 마스킹)**: 현재 이미지 단위의 비식별화 품질이 안정화되면, 차후에는 **동영상(Video)에서 프레임별로 개인정보를 찾아 지금과 동일한 수준으로 마스킹하는 영상 비식별화 시스템**으로 확대 적용할 예정.
